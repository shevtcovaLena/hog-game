# Найди предметы 🎯

Игра на PixiJS: ищи предметы на картинке, зумь, двигай камеру.

🔗 [Демо на Vercel](https://hog-game-nu.vercel.app)

## Запуск

```bash
npm install
npm run dev
npm run build
```

| Действие | Управление |
|----------|-----------|
| Двигать камеру | Drag мышкой или тач |
| Зум | Pinch / колесо / кнопки ➕➖ |
| Сброс вида | Кнопка 🔳 |

## Технологии
- TypeScript
- PixiJS + pixi-viewport
- Vite
